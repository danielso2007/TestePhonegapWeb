Example

Watching geolocation...

Relógios para alterações à posição atual do dispositivo.

Parâmetros
geolocationSuccess : O retorno de chamada que é passado da posição atual.
geolocationError : (Opcional) O retorno de chamada que executa se ocorrer um erro.
geolocationOptions : (Opcional) As opções de geolocalização.

Devoluções
String: retorna um ID de relógio que faz referência o intervalo de posição de observação. O ID de relógio deve ser usado com geolocation.clearWatch parar de assistir a mudanças na posição.

Descrição
geolocation.watchPosition é uma função assíncrona. Ele retorna a posição atual do dispositivo, quando uma mudança de posição é detectada. Quando o dispositivo recupera um novo local, o geolocationSuccess callback executa com uma Position de objeto como parâmetro. Se houver um erro, o geolocationError de retorno é executado com um PositionError objecto como o parâmetro.

GeolocationOptions

Parâmetros opcionais para personalizar a recuperação da geolocalização Position.

{ maximumAge: 3000, timeout: 5000, enableHighAccuracy: true };

Opções

  • enableHighAccuracy: fornece um indício de que o aplicativo precisa os melhores resultados possíveis. Por padrão, o dispositivo tenta recuperar a Position através de métodos baseados em rede. Definir essa propriedade como true conta a estrutura para usar métodos mais precisos, como o posicionamento por satélite. (Boolean)

  • timeout: O comprimento máximo de tempo (milissegundos) que é permitida a passagem da chamada para geolocation.getCurrentPosition ou geolocation.watchPosition até o correspondente geolocationSuccess callback executada. Se o geolocationSuccess callback não é invocado dentro deste tempo, a geolocationError callback é passado um PositionError .TIMEOUT código de erro. (Note-se que, quando usado em conjunto com geolocation.watchPosition , o geolocationError de retorno poderia ser chamada em cada intervalo timeout milissegundos!) (Número)

  • maximumAge: Aceitar uma posição em cache cuja idade não é maior do que o tempo especificado em milissegundos (Number)
