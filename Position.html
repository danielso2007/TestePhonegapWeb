PositionError

A PositionError object is passed to the geolocationError callback when an error occurs.

Properties

  • code: Um dos códigos de erro predefinidos listados abaixo..

  • message: Mensagem de erro descrevendo os detalhes do erro encontrado.

Constants

  • PositionError.PERMISSION_DENIED
  • PositionError.POSITION_UNAVAILABLE
  • PositionError.TIMEOUT

Description

The PositionError objeto é passado para o geolocationError função callback quando ocorre um erro com geolocalização. Possui os seguintes códigos de erro:

  • PositionError.PERMISSION_DENIED: Devolvido quando os usuários não permitir que o aplicativo recupere informação sobre a posição. Isto é dependente da plataforma..

  • PositionError.POSITION_UNAVAILABLE: Retornado quando o dispositivo não é capaz de recuperar a posição. Em geral, isso significa que o dispositivo não está conectado a uma rede ou não pode obter uma correção por satélite.

  • PositionError.TIMEOUT: Retornado quando o dispositivo não é capaz de recuperar uma posição dentro do tempo especificado pelo timeout incluído no timeout included in geolocationOptions. Quando usado com geolocation.watchPosition, este erro pode ser passado várias vezes para o to the geolocationError callback every timeout milliseconds.
